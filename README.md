# AI Network Threat Predictor

A real-time network threat detection system built using Python, Scapy, and Machine Learning (RandomForestClassifier).

## Features
- Real-time packet capture and logging
- Automatic network interface detection
- AI model predicts threat level per packet
- Domain and process mapping for each packet
- Optional dashboard visualization

## Tech Stack
Python, Scapy, Pandas, psutil, joblib, Chart.js, HTML
